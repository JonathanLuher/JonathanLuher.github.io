Compiladores


Traductor

En este programa se hace uso del lenguaje de python para crear un traductor mediante dos listas guardadas en docmentos de texto, en el que además puede mostrar el diccionario que se tiene del ingles al español. También te corrige si tienes un error de escritura mediante el algoritmo de Jaccard.

Información detallada

Detector de numeros

Proyecto elaborado en python que tiene como objetivo detectar numeros dibujados por el usuario. El objetivo fue logrado entrenando el programa mediante un perceptron, asignando pesos segun las imagenes con las que fue entrenada. También incluye otras dos versiones mas simples.

Información detallada

Generador de texto

El objetivo de este proyecto es generar texto a partir de otro texto introducido por el usuario, el programa se encargá de obtener una lista de palabras, después de obtener la matriz de probabilidades, para posteriormente generar texto usando el metodo de caminata de Carchoff.

Gramatica de C

La elaboración de este proyecto se hizo mediante varias nmetodologías para poder detectar que un codigo en C se ecuentre sintacticamente correcta antes de compilarse, verificando que las llaves se encuentren cerradas, además que no falten ";" al finalizar cada linea (si es que no hay una llave).
